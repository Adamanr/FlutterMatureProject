# Matule 2026 – Storybook

Отдельное приложение для демонстрации всех компонентов из **ui_kit**  
(Модуль B – Дизайн-система и UI-kit)

## Цель
Демонстрация всех состояний UI-компонентов экспертам WorldSkills без запуска основного приложения.

## Как запустить

```bash
# Из корня проекта (melos обязателен!)
melos run storybook

# Или напрямую
flutter run --project-directory=packages/storybook
